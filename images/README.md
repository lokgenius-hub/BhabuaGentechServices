# Images Folder

This folder is for storing website images.

## Recommended Images to Add:

1. **Logo**: Company logo (SVG or PNG format)
   - `logo.png` or `logo.svg`
   - Recommended size: 200x60px (or similar aspect ratio)

2. **Favicon**: Website favicon
   - `favicon.png`
   - Recommended size: 32x32px or 64x64px

3. **Service Images**: Visual representation of services
   - `school-management.jpg`
   - `marriage-hall.jpg`
   - `shop-billing.jpg`
   - `personal-website.jpg`
   - Recommended size: 800x600px

4. **About Section**: Company photos, team photos
   - `office.jpg`
   - `team.jpg`
   - Recommended size: 1200x800px

5. **Hero Section**: Hero image or illustration
   - `hero-image.png` or `hero-image.svg`
   - Recommended size: 1000x1000px

## Image Optimization Tips:

- Use WebP format for better compression
- Compress images before uploading (use TinyPNG.com)
- Use descriptive file names (e.g., `school-management-system.jpg`)
- Maintain aspect ratios for consistency
- Use SVG for logos and icons when possible

## File Naming Convention:

- Use lowercase
- Use hyphens for spaces (not underscores)
- Be descriptive
- Examples: `team-photo.jpg`, `service-icon-1.png`
